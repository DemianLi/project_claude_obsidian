## 快速定位規則（Claude 使用）

```
任務關鍵字 → 對應模組：

若任務包含「登入/登出/帳號/密碼/權限/角色」
  → 讀 functional/{auth}.md

若任務包含「訂單/出貨/退款/取消/購物車」
  → 讀 functional/{order}.md

若任務包含「報表/匯出/統計/圖表」
  → 讀 functional/{report}.md

若任務跨多個模組（例：「修改訂單後更新庫存並通知用戶」）
  → 分別讀取各相關模組的 functional/*.md

若任務關鍵字無法對應任何模組
  → 告知使用者：「無法從任務描述定位模組，請說明這個功能屬於哪個業務域」
```
