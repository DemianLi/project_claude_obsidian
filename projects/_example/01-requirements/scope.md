## 📌 Claude 使用規則

當甲方提出新需求時：

```
1. 查詢此文件確認是否已在合約範圍內
   ├─ 是 → 正常 /ingest 流程
   └─ 否或不確定 → 告知使用者：
      「此需求不在原始合約範圍（v1.0）內，建議執行 /cr 建立變更請求，
       評估影響範疇後再決定是否追加報價。」

2. 執行 /cr 後，等待使用者決定：
   ├─ 議價成功 → 加入追加範疇表，需求進入對應 functional/{module}.md（標記 CR 版本）
   └─ 不接受 → 加入超出範疇表，需求不進入知識庫
```
