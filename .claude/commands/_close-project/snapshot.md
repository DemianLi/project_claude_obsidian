# Phase 2：生成交付文件快照

詢問使用者：

```
① 是否已生成甲方交付文件？(Y / 需要現在生成)
   → 若「需要現在生成」：執行 /export --client

② 是否需要生成技術交接文件？(Y / 不需要 / 需要現在生成)
   → 若「需要現在生成」：執行 /export --handover

③ 最終版本號？（例：v1.2.0）
   → 用於 CHANGELOG 的最終版本標記
```

確認後記錄以下資訊供後續 Phase 使用：
- 交付文件路徑（若已生成）
- 技術交接文件路徑（若已生成）
- 最終版本號

→ 完成後讀取 `_close-project/changelog.md`
