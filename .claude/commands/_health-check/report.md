## 快速修復建議（Claude 主動提供）

| 問題 | 快速修復 |
|------|----------|
| REQ 缺少驗收條件 | 「要我根據需求描述幫你補上驗收條件嗎？」 |
| ADR 缺少後果 section | 「要我根據現有內容推斷後果並補充嗎？」 |
| PENDING 超過 60 天 | 「要我整理成可以直接寄給甲方的 email 嗎？」 |
| PAT 缺少 tags | 「要我根據模式內容自動推薦 tags 嗎？」 |

append 記錄至 `wiki/log.md`。
